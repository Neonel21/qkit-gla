Welcome to Qkit’s documentation!

This is the documentation for the Qkit package modified at Quantum Circuits Group at the University of Glasgow.

What is Qkit? Qkit is a Python package for quantum computing experiments. It provides a set of tools for the design, simulation, and measurement of experiements at UofG.

To install Qkit, simply run:

pip install qkit

Requirements

Before you install Qkit, note that you’re in the right environment. Qkit requires Python 3.6 or later. If you’re not sure which version of Python you have, run:

python --version

On a standard experimental PC, create a conda environment with Python 3.6 or later:

conda create -n qkit-gla python=3.7

Then, activate the environment:

conda activate qkit-gla

Finally, install Qkit:

pip install qkit
